Next Steps

With the new confidence, I did some planning and rearranging on the board. The next thing will be a bigger jump with “render thumbnails for pictures”. This will make the gallery more presentable and start approaching something that actually works. I’m planning on following other project’s lead and use rmagic for the image manipulation. The goal is for each image in a gallery render out a square thumbnail in a (configurable??) size and attach enough information about that thumbnail to the images information so that the layout can use the thumbnail as link to the actual picture. This should end up very similar to how the current gallery looks like.

While I’m now confident about the basics, I’m wondering how the thumbnail rendering will fit into the incremental rendering. I’ve stumbled over Drops while browsing through the object tree, which seem to represent the target file before it is written? Tune in on the weekend to see me figure it out, or fail hilariously!
